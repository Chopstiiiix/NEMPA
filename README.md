# NEMPA

**Nigerian Emergency Missing Person Alert** — community safety alerts for missing persons and robbery incidents, with location-based push notifications.

React + Vite + TypeScript · Capacitor (Android/iOS) · Supabase (Postgres + PostGIS) · Firebase Cloud Messaging.

## Quick start
```bash
npm install
cp .env.example .env   # add your Supabase URL + anon key
npm run dev
```

See **CLAUDE.md** for full setup (Supabase schema, Capacitor native, FCM) and the build roadmap.

> Not affiliated with any government AMBER Alert system. A community tool — all alerts are moderated before broadcast.
